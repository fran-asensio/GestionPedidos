# Proyecto Pedidos

## Descripción
Proyecto Pedidos es una aplicación desarrollada en Java que permite gestionar de forma eficaz pedidos, clientes y productos. Su objetivo es facilitar la organización y el control de estos elementos mediante una interfaz intuitiva y herramientas funcionales.

## Características principales
- Creación, edición y eliminación de pedidos.
- Gestión de la información de clientes y productos.
- Generación de reportes para analizar los datos de pedidos y estadísticas.

## Tecnologías utilizadas
Este proyecto está desarrollado utilizando Java, con una base de datos para almacenar la información y Git para el control de versiones.

## Instalación y uso
1. Clonar el repositorio del proyecto en tu máquina local.
2. Importar el proyecto a tu IDE de preferencia (NetBeans, Eclipse, IntelliJ, etc.).
3. Configurar la conexión a la base de datos en el archivo correspondiente.
4. Compilar y ejecutar la aplicación.
